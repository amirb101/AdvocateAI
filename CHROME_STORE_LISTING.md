## Detailed Description

### What is Advocate LLM?

Advocate LLM helps you become a more rounded thinker by automatically finding polarising claims in news articles and showing balanced counterarguments directly in the page.

### How It Works

1. **Read an article** - Navigate to any news article
2. **Click the extension** - Press "Find 5 Polarising Points"
3. **See highlights** - Polarising quotes are highlighted in yellow
4. **Hover for counterpoints** - View counterarguments, explanations, and sources

### Key Features

✅ **Automatic Detection** - Finds polarising claims automatically
✅ **In-Context Counterpoints** - Shows opposing views right where you're reading
✅ **Multiple LLM Support** - Works with OpenAI, DeepSeek, or Anthropic
✅ **Privacy-Focused** - Article text processed locally before analysis
✅ **Free Tier** - 5 analyses per day, resets daily

### Why Use Advocate LLM?

- **Challenge Bias** - See multiple perspectives on controversial topics
- **Think Critically** - Understand why claims are polarising
- **Stay Informed** - Get balanced viewpoints without leaving the article
- **Learn** - Understand different sides of complex issues

### Privacy

- Article text processed locally
- No tracking or analytics
- API keys stored securely (if using backend)
- See our Privacy Policy for details

### Getting Started

1. Install the extension
2. Configure your backend URL (or use direct API with your own key)
3. Visit a news article
4. Click the extension icon
5. Click "Find 5 Polarising Points"
